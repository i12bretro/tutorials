Setup an Official Windows 10 VM in Under 10 Minutes

  1. Open a browser and navigate to https://developer.microsoft.com/en-us/microsoft-edge/tools/vms/
  2. Select VirtualBox from the platform options
  3. Extract the downloaded .zip file
  4. Launch VirtualBox
  5. Select File > Import Appliance...
  6. Click the Browse icon > Navigate to the extracted files .ova file > Select MSEdge - Win10.ova
  7. Click Next and then Import
  8. Make sure the MSEdge - Win10 VM is selected and click Start > Normal
  9. That's it
