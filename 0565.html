Running a Windows Server 2022 VM in Virtualbox

  1. Download the Windows Server 2022 iso Download
  2. Launch Virtualbox
  3. Create a New VM by selecting Machine > New

    Name: Windows Server 2022
    Machine Folder: C:\VMs
    Type: Microsoft Windows
    Version: Other Windows (64-bit)
    Memory Size: 4096 MB
    Hard disk: Create a virtual hard disk now

  4. Click Create
  5. On the Create Virtual Hard Disk dialog

    Name the virtual disk image Windows Server 2022.vdi
    File size: 50.00 GB
    Hard disk file type: VDI
    Storage on physical hard disk: Dynamically Allocated

  6. Click Create
  7. Select the VM and Click Settings
  8. Select System > Processor
  9. Give the VM at least 2 processors
  10. Select Display
  11. Slide the Video Memory to 128 MB
  12. Select Storage
  13. Click on the CD-ROM drive
  14. Select the disc dropdown to the right > Choose a virtual optical disc file...
  15. Browse to and select the Windows Server 2022 .iso file
  16. Click OK
  17. Make sure the Windows Server 2022 VM is selected and click Start > Normal
  18. Set the Language and Keyboard preferences > Click Next
  19. Click Install Now
  20. Click I don't have a product key at the bottom
  21. Select the desired edition > Click Next
  22. Check the I accept box > Click Next
  23. Click Custom (advanced) > Click Next
  24. Wait for Windows to copy files, the VM will reboot several times
  25. Enter a password > Re-type the password > Click Finish
  26. Select Input > Keyboard > Ctrl-Alt-Del
  27. Input the administrator password and login
  28. Welcome to Server 2022

Optional Steps

  1. Select Devices > Insert Guest Additions CD image...
  2. If VirtualBox Guest Additions doesn't autorun, open File Explorer > CD Drive > Right click VBoxWindowsAdditions.exe > Run as administrator
  3. Install Guest Additions accepting the defaults
  4. Accept any driver popups from the Guest Additions installer by clicking the Install button
  5. After the installation completes select Reboot now > Click Finish
