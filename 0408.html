Install Mouse Toggle on FireTV Device

  1. Install the Downloader application
  2. Launch Downloader
  3. Navigate to https://tv.aptoide.com/
  4. Click Download button
  5. Click the Download APK button
  6. A prompt will display to go to settings and allow installing unknown apps
  7. Click Settings > Install Unknown Apps > Downloader > Toggle the setting to On
  8. Click back and toggle the ADB Debugging setting On
  9. Click back a few times to return to the Downloader app
  10. Click Install and then Install again
  11. After the app installation completes click Open
  12. Use the up arrow on the remote to select the search icon
  13. Search for Mouse Toggle
  14. Scroll through the results until you reach Mouse Toggle for Fire TV
  15. Click the Install button
  16. Once the app installation completes click Open
  17. Make sure the Enable the Mouse Service and Auto start the mouse service on device start-up are both enabled
  18. Reboot the FireTV device by going to Settings > My Fire TV > Restart and select Restart again to confirm
  19. After the Fire TV device has booted back up wait a few moments and an ADB confirmation prompt should display
  20. Check the Always allow from this computer option and click OK
  21. At this point, you can press the Play/Pause button the Fire TV remote twice to toggle mouse mode back and forth
