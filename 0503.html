Booting TWRP Recovery on Amlogic S905w Android TV Box

Things You Will Need

  • A microSD card at least 4 GB https://amzn.to/36CpShC | https://amzn.to/3CXJngt | https://amzn.to/3JqyxSP

Booting TWRP

  1. Download TWRP recovery for Amlogic S905w devices Download
  2. Rename the downloaded file recovery.img
  3. Format a microSD card to FAT32
    1. Right Click Start > File Explorer
    2. Right Click the microSD card > Format...
    3. Select FAT32 from the File System dropdown
    4. Make sure Quick Format is checked
    5. Click Start
  4. Copy recovery.img to the root of a microSD card
  5. Safely eject the microSD card
  6. Unplug power from the TV box
  7. Plug the microSD card into the Amlogic S905w based TV box
  8. While holding the reset button on the TV box (sometimes hidden inside the AV port) with a toothpick or unfolded paperclip, plug in the power cord
  9. Continue holding the reset button for 10-15 seconds
  10. TWRP recovery should load
