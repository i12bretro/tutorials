Setup the Official NextCloud VM in VirtualBox in Under 10 Minutes

  1. Download Nextcloud-VM.ova Download
  2. Launch VirtualBox
  3. Select File > Import Appliance...
  4. Click the choose file icon and navigate to the downloaded Nextcloud-VM.ova
  5. Change the VM Name to NextCloud
  6. Click the Import button
  7. After the import process completes, select the NextCloud VM and click Start > Normal Start
