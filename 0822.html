What's New in the Upcoming VirtualBox 7

What is VirtualBox?

VirtualBox is a powerful x86 and AMD64/Intel64 virtualization product for enterprise as well as home use. -https://www.virtualbox.org/

Key New Features and Improvements

  • Dark mode!!
  • Real-time guest VM activity overview/resource monitor
  • Integrated host to guest file manager
  • More detailed Guest Version dropdown
  • Ability to set an iso image during VM creation
    • If the OS on the .iso is detected, ability to install unattended
    • Ability to install guest additions unattended
  • Updated BIOS boot logo
  • Better integrated Virtual Media Manager
  • ...a list of official changes can be found on page 413 of the User Manual
