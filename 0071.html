Running a Windows 2000 VM in Virtualbox

  1. Download Microsoft Windows 2000 Download
  2. Extract the Windows 2000 files
  3. Launch Virtualbox
  4. Create a New VM by selecting Machine > New

    Name: Windows 2000
    Machine Folder: C:\VMs
    Type: Microsoft Windows
    Version: Windows 2000
    Memory Size: 1024 MB
    Hard disk: Create a virtual hard disk now

  5. Click Create
  6. On the Create Virtual Hard Disk dialog

    Name the virtual disk image Windows 2000.vdi
    File size: 4.00 GB
    Hard disk file type: VDI
    Storage on physical hard disk: Dynamically Allocated

  7. Click Create
  8. Select the VM and Click Settings
  9. Select Storage
  10. Click on the CD-ROM drive
  11. Select the disc dropdown to the right > Choose a virtual optical disc file...
  12. Browse to the extracted Windows 2000 files and select Win2000 .iso
  13. Click OK
  14. Make sure the Windows 2000 VM is selected and click Start > Normal
  15. The Windows 2000 setup should load on boot
  16. Press Enter to Continue with Setup 
  17. Press C to Continue with Setup
  18. Press F8 to agree to the Licensing Agreement
  19. Press Enter to Install to the unpartitioned space
  20. Select the top option to format using the NTFS file system
  21. Press Enter to continue
  22. Setup will copy files and then reboot the VM
  23.  When the VM boots back up, Click Next
  24.  Wait for Windows to install devices and drivers
  25.  Click Next
  26. Enter a name and organization > Click Next
  27. Enter a valid Windows 2000 product key ( XGH9J-KBPBD-FXDKQ-K36XB-X9J6Y),  click Next
  28. Give the computer a name
  29. Enter an administrator password > Re-type to confirm > Click Next
  30. Set the date/time and select a timezone > Click Apply > Click OK
  31.  Select Typical settings on the Network settings dialog > Click Next
  32.  Select No, this computer is not on a network > Click Next 
  33. Wait for Windows to install components
  34. Click Finish, the VM will reboot 
  35. On the Network Identification dialog select This computer is for home use... > Click Next > Click Next > Click Finish, the VM will reboot 
  36. Welcome back to Windows 2000...Y2k
  37. Revel in the nostalgia

Optional Steps

  1. Select Devices > Insert Guest Additions CD image...
  2. If VirtualBox Guest Additions doesn't autorun, open My Computer > CD Drive > Double click VBoxWindowsAdditions-x86.exe
  3. Install Guest Additions accepting the defaults
  4. Accept any driver popups from the Guest Additions installer by clicking the Yes button
  5. After the installation completes select Reboot now > Click Finish
  6. Upon rebooting, you can now full screen the VM and the interal resolution will automatically change to match the window size
  7. Download i12bretro Windows SP.iso Download
  8. Select Devices > Optical Drives > Select disk image...
  9. Navigate to and select i12bretro_Windows_SP.iso
  10. If the VM doesn't autorun, open My Computer > CD Drive
  11. Open the Windows 2000 folder > Double click Windows 2000 SP4.exe
  12. Follow the dialogs to install Service Pack 4
  13. Click Finish to reboot and complete the SP4 install
