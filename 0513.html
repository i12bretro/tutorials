Run TWRP Recovery on Allwinner H6 (Tanix TX6) Android TV Box

NOTE: In my experience, this did not overwrite the stock recovery and I was unable to get the System partition to mount

Things You Will Need

  • A microSD card at least 4 GB https://amzn.to/36CpShC | https://amzn.to/3CXJngt | https://amzn.to/3JqyxSP

Preparation and Booting Stock Recovery

  1. Download TWRP recovery for Tanix TX6 Download
  2. Format a microSD card to FAT32
    1. Right Click Start > File Explorer
    2. Right Click the microSD card > Format...
    3. Select FAT32 from the File System dropdown
    4. Make sure Quick Format is checked
    5. Click Start
  3. Copy the downloaded .zip file to the microSD card
  4. Safely eject the microSD
  5. Unplug power from the TV box
  6. Plug the microSD card into the Tanix TX6 TV box
  7. While holding the reset button on the TV box (on the back beside the two USB ports) with a toothpick or unfolded paperclip, plug in the power cord
  8. Continue holding the reset button for 10-15 seconds
  9. The stock recovery should load

Running TWRP Recovery

  1. With the device in stock recovery, select Appy update from SD card from the main menu
  2. Select the downloaded TWRP_TX6.zip
  3. TWRP recovery should load
