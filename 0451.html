Booting TWRP

  1. Download TWRP recovery for Amlogic devices Download
  2. Rename the downloaded file recovery.img
  3. Format a microSD card to FAT32
    1. Right Click Start > File Explorer
    2. Right Click the microSD card > Format...
    3. Select FAT32 from the File System dropdown
    4. Make sure Quick Format is checked
    5. Click Start
  4. Copy recovery.img to the root of a microSD card
  5. Safely eject the microSD card
  6. Unplug power from the TV box
  7. Plug the microSD card into the Amlogic based TV box
  8. While holding the reset button on the TV box (sometimes hidden inside the AV port) with a toothpick or unfolded paperclip, plug in the power cord
  9. Continue holding the reset button for 10-15 seconds
  10. TWRP recovery should load

Installing TWRP (optional)

Only install TWRP if it is required by a custom ROM, otherwise, booting to it from microSD when needed will suffice

  1. With the device in TWRP recovery, select Install from the main menu
  2. Select Install Image from the lower right
  3. Select Storage from the lower right > Select MicroSD Card > Click OK
  4. Select recovery.img from the listing
  5. Select Recovery from the partition listing
  6. Click and drag the slider to the right to begin flashing
  7. The flash should complete in just a few seconds
  8. Once completed successfully, power off the Android TV box
  9. Remove the microSD card
  10. While holding the reset button, power the TV box back on
  11. Continue holding the reset button for 10-15 seconds
  12. TWRP recovery should load, this time from the recovery partition
