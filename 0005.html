Customizing Apache HTTPD Error Pages

  1. Download Apache HTTPD Download
  2. Install Apache HTTPD
  3. Navigate to <ApacheInstallDirectory>/error
  4. Edit top.html and bottom.html to your liking
  5. Save top.html and bottom.html
  6. Open a web browser
  7. Browse to a local URL that is invalid to force a 404 error from Apache HTTPD Forced 404
