Running a Vanilla ChromiumOS VM in Virtualbox

NOTE: This method uses an older outdated version of ChromiumOS

  1. Open a web browser and download the ChromiumOS .ova Download
  2. Extract the downloaded .zip file
  3. Launch VirtualBox
  4. Create a New VM by selecting Machine > New

    Name: ChromiumOS
    Machine Folder: C:\VMs
    Type: Linux
    Version: Other Linux (32-bit)
    Memory Size: 2048 MB
    Hard disk: Use an existing virtual hard disk now

  5. Click the folder icon > Browse to and select the extracted ChromeOS-Vanilla .vdi file
  6. Click Create
  7. Select the VM and Click Settings
  8. Select System > Processor
  9. Check the Enable PAE/NX checkbox
  10. Select Display
  11. Slide the Video Memory to 128 MB
  12. Select Network
  13. Set the attached to dropdown to Bridged Adapter
  14. Expand the Advanced options area
  15. Set the Adapter Type to Intel PRO/1000 MT Desktop
  16. Click OK
  17. Make sure the ChromiumOS VM is selected and click Start > Normal
  18. Click Continue on the Welcome! screen
  19. Press Escape to cancel checking for updates
  20. Click the browse as guest link
  21. Welcome to ChromiumOS
