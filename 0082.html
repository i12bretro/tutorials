Running a Windows Server 2008 VM in Virtualbox

  1. Locate a Windows 2008 iso
  2. Launch Virtualbox
  3. Create a New VM by selecting Machine > New

    Name: Windows Server 2008
    Machine Folder: C:\VMs
    Type: Microsoft Windows
    Version: Windows 2008 (64-bit)
    Memory Size: 4096 MB
    Hard disk: Create a virtual hard disk now

  4. Click Create
  5. On the Create Virtual Hard Disk dialog

    Name the virtual disk image Windows Server 2008.vdi
    File size: 40.00 GB
    Hard disk file type: VDI
    Storage on physical hard disk: Dynamically Allocated

  6. Click Create
  7. Select the VM and Click Settings
  8. Select System > Processor
  9. Give the VM at least 2 processors
  10. Select Display
  11. Slide the Video Memory to 128 MB
  12. Select Storage
  13. Click on the CD-ROM drive
  14. Select the disc dropdown to the right > Choose a virtual optical disc file...
  15. Browse to and select the Server 2008 .iso file
  16. Click OK
  17. Make sure the Server 2008 VM is selected and click Start > Normal
  18. Click Next
  19. Click Install Now
  20. Select the version to install > Click Next
  21. Check the I accept box, Click Next
  22. Click Custom (advanced) > Click Next
  23. Wait for Windows to copy files, the VM will reboot
  24. Enter an Administrator password > Re-type the password > Click the Next Arrow
  25. Click OK
  26. Welcome back to Windows Server 2008

Optional Steps

  1. Select Devices > Insert Guest Additions CD image...
  2. If VirtualBox Guest Additions doesn't autorun, open My Computer > CD Drive > Double click VBoxWindowsAdditions-amd64.exe
  3. Install Guest Additions accepting the defaults
  4. Accept any driver popups from the Guest Additions installer by clicking the Install button
  5. After the installation completes select Reboot now > Click Finish
  6. Upon rebooting, you can now full screen the VM and the internal resolution will automatically change to match the window size
  7. Log back into the VM
  8. Download i12bretro Windows SP.iso Download
  9. Select Devices > Optical Drives > Select disk image...
  10. Navigate to and select i12bretro_Windows_SP.iso
  11. If the VM doesn't autorun, open My Computer > CD Drive
  12. Open the Server 2008 & Windows 7 folder > Double click SP1 x64.exe
  13. Follow the dialogs to install Service Pack 1
  14. Click Finish to reboot and complete the SP1 install
